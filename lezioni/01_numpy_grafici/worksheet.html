Copia ed esegui
⏱ 0 – 10 min

Scrivi esattamente questo codice nel tuo editor e mandalo in esecuzione. Osserva il grafico prodotto.

grafico_funzioni.py
import matplotlib.pyplot as plt
import numpy as np

plt.figure('tre funzioni', [5, 3.5])

x = np.linspace(0, 5, 30) # vettore di NumPy

plt.plot(x, x, label='lineare', linewidth=2)
plt.plot(x, x**2, label='quadratica', linewidth=2)
plt.plot(x, x**3, label='cubica', linewidth=2)

plt.title('Grafico di tre funzioni con NumPy')
plt.legend()
plt.grid(True, color='grey')
plt.show()
